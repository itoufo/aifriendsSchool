### 1.1 Chain of Thought（思考の連鎖）実践

**Chain of Thought（CoT）** は、AIに段階的に考えさせることで、複雑な問題の回答精度を大幅に向上させるテクニックです。

#### 基本形と応用形の比較

**悪い例（単純な質問）:**
```
この商品の売上を分析してください。
```

**良い例（CoTを活用）:**
```
以下の商品売上データを分析してください。

## 分析の進め方
以下のステップで段階的に分析を進めてください：

### Step 1: データの概要把握
- データの期間、レコード数、含まれる項目を確認
- 欠損値や異常値の有無をチェック

### Step 2: 基本統計の算出
- 売上の合計、平均、中央値、標準偏差
- カテゴリ別の売上構成比

### Step 3: 時系列トレンド分析
- 月別・週別の売上推移
- 前年同期比較（データがある場合）
- 季節性パターンの有無

### Step 4: 深掘り分析
- 上位20%の商品が占める売上割合（パレート分析）
- 売上に影響を与える要因の仮説

### Step 5: 示唆と提言
- データから読み取れる3つの重要な発見
- 売上向上のための具体的なアクション提案

各ステップで発見したことを明記し、次のステップに進んでください。
```

### 1.2 Few-shot Learning の設計

**Few-shot Learning** は、AIに例を示すことで、望む出力形式や品質を学習させるテクニックです。

#### 効果的なFew-shot設計の3原則

1. **多様性**: 異なるパターンの例を含める
2. **一貫性**: 出力形式を統一する
3. **境界ケース**: 難しいケースも例示する

### 1.3 システムプロンプトの最適化

**システムプロンプト** は、AIの「人格」「専門性」「制約」を定義する重要な要素です。

#### プロフェッショナルなシステムプロンプトの構成要素

```
# システムプロンプト設計テンプレート

## 1. 役割定義（Role）
あなたは[専門分野]の専門家です。
[X]年の実務経験があり、[具体的な実績]を持っています。

## 2. 目的（Objective）
あなたの目的は、[ユーザーが達成したいこと]を支援することです。

## 3. スタイル（Style）
- トーン: [フォーマル/カジュアル/専門的]
- 文体: [簡潔/詳細/会話的]
- 言語: 日本語

## 4. 制約（Constraints）
- [してはいけないこと1]
- [してはいけないこと2]
- [必ず含めるべきこと]

## 5. 出力形式（Format）
回答は以下の形式で提供してください：
[具体的なフォーマット指定]

## 6. 例外処理（Edge Cases）
- 情報が不足している場合: [対応方法]
- 専門外の質問の場合: [対応方法]
```

### 3.1 Zapierの基本設定

**Zapier** は、異なるWebサービスを連携させる自動化ツールです。

#### Zapierの基本用語
- **Zap**: 自動化ワークフローのこと
- **Trigger**: ワークフローを開始するきっかけ
- **Action**: トリガー後に実行される処理
- **Multi-Step Zap**: 複数のアクションを連続実行

#### 無料プランでできること
- 5つのZap
- 100タスク/月
- 2ステップのZap（Trigger + 1 Action）

### 3.2 Gmail→AI分析→Slack通知の構築

**ユースケース:** 重要なメールが来たら、AIで内容を要約してSlackに通知

```
┌─────────────────┐
│ Gmail          │
│ 新着メール受信  │ ← Trigger
└────────┬────────┘
         ↓
┌─────────────────┐
│ Filter         │
│ 特定条件で絞込  │
└────────┬────────┘
         ↓
┌─────────────────┐
│ OpenAI/ChatGPT │
│ メール要約生成  │ ← Action 1
└────────┬────────┘
         ↓
┌─────────────────┐
│ Slack          │
│ チャンネル投稿  │ ← Action 2
└─────────────────┘
```

#### Step-by-Step 構築手順

**Step 1: Zapierアカウント設定**
1. https://zapier.com でアカウント作成
2. 「Create Zap」をクリック

**Step 2: トリガー設定（Gmail）**
1. App: Gmail を選択
2. Event: New Email を選択
3. Gmailアカウントを接続
4. フォルダを「受信トレイ」に設定
5. テスト実行で最新メールを取得

**Step 3: フィルター設定（任意）**
1. 「+」をクリックしてステップ追加
2. Filter by Zapier を選択
3. 条件例: From contains "important-client.com"

**Step 4: AI分析設定（OpenAI）**

## 修了プロジェクト

### 課題: 自分または知人の業務を1つAIで自動化し、効果を測定する

このLevel 2の総まとめとして、**実際の業務**をAIで改善するプロジェクトに取り組みます。

#### プロジェクト要件

```
## 対象業務
- 自分の本業または副業で行っている定型業務
- 知人や家族の業務（許可を得ること）
- 週1回以上発生し、30分以上かかる業務

## 自動化の範囲
以下のいずれかを含むこと：
- AIによる文章生成/要約
- AIによるデータ分析
- Zapier/Make による自動化
- ChatGPT/Claude の定型プロンプト化

## 成果物
1. 業務改善レポート（テンプレート後述）
2. 作成したプロンプト/ワークフロー
3. Before/After の比較データ
```

#### 提出フォーマット

#### 評価基準

| 評価項目 | 配点 | 評価ポイント |
|----------|------|--------------|
| 課題設定 | 20点 | 実務的で改善価値のある業務を選んでいるか |
| 解決策 | 30点 | AIツールを適切に活用しているか |
| 効果測定 | 25点 | 定量・定性両面で効果を測定しているか |
| 実行力 | 15点 | 実際に運用して改善を確認しているか |
| レポート | 10点 | 分かりやすく構成されているか |

**合格ライン:** 70点以上
