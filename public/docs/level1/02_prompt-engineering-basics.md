# プロンプトエンジニアリング入門

**学習時間**: 2週目（約10時間）  
**対象**: AI基礎知識のある初心者  
**目標**: 効果的なプロンプト設計技術を身につけ、AIとの対話を最適化する

## 概要

プロンプトエンジニアリングは、AIから最適な回答を引き出すためのコミュニケーション技術です。適切なプロンプト設計により、AIの能力を最大限に活用し、業務効率を大幅に向上させることができます。この章では、基本的なプロンプト設計から目的別の実践テクニックまでを体系的に学習します。

## 学習項目

### 2-1: 基本プロンプト設計

#### 学習目標
- 良いプロンプトの5要素（役割、指示、文脈、制約、出力形式）を理解する
- ChatGPT、Gemini、Claudeの特性に応じたプロンプト設計ができる
- ペルソナ設定による回答品質の向上を実現する

#### 良いプロンプトの5要素

**1. 役割（Role）**
AIに特定の専門家や立場を演じさせる

*例:*
```
あなたは経験豊富なマーケティングコンサルタントです。
```

**2. 指示（Instruction）**
具体的で明確なタスクの説明

*例:*
```
新商品のマーケティング戦略を立案してください。
```

**3. 文脈（Context）**
タスクに必要な背景情報

*例:*
```
商品：オーガニック化粧品
ターゲット：30-40代女性
予算：500万円
期間：3ヶ月
```

**4. 制約（Constraints）**
条件や制限事項の明示

*例:*
```
- 既存ブランドとの競合を避ける
- 環境配慮をアピールポイントとする
- SNS活用を重視する
```

**5. 出力形式（Output Format）**
希望する回答の構造や形式

*例:*
```
以下の形式で回答してください：
1. 戦略概要（200字以内）
2. 具体的施策（3つ以上）
3. 期待効果と測定指標
4. リスクと対策
```

#### プロンプト設計のベストプラクティス

**明確性（Clarity）**
- 曖昧な表現を避ける
- 具体的な数値や条件を含める
- 一つのプロンプトで一つのタスク

*悪い例:*
```
良い文章を書いて
```

*良い例:*
```
新入社員向けの「効果的なメールの書き方」について、
以下の構成で800字程度の文章を作成してください：
1. 件名の付け方（3つのポイント）
2. 本文の構造（結論先行型）
3. 注意すべきマナー（5項目）
```

**段階的指示（Step-by-step）**
複雑なタスクは段階に分けて指示

*例:*
```
以下の手順で分析を行ってください：
1. まず、データの概要を把握する
2. 次に、トレンドを特定する
3. 最後に、課題と改善案を提示する
```

#### 主要AIツールの特性理解

**ChatGPT (OpenAI)**
- 強み：創造性、対話の自然さ
- 適用：文章作成、アイデア発想、コーディング
- 注意：情報の正確性確認が必要

**Gemini (Google)**
- 強み：最新情報への接続、多言語対応
- 適用：調査、翻訳、Googleサービス連携
- 注意：企業情報の取り扱い

**Claude (Anthropic)**
- 強み：長文処理、安全性、論理的思考
- 適用：文書分析、要約、倫理的な判断
- 注意：創造性では他モデルに劣る場合がある

### 2-2: 目的別プロンプト実践

#### 学習目標
- 情報収集、要約、翻訳、アイデア出し、文章校正の各用途に適したプロンプトを作成できる
- Perplexity AI、Elicitなどの専門ツールを活用できる
- 効果的な情報検索戦略を実行できる

#### 1. 情報収集プロンプト

**基本テンプレート:**
```
【役割】情報収集の専門家として
【タスク】[テーマ]に関する最新情報を収集し、整理してください
【条件】
- 信頼できるソースからの情報
- 2023年以降の情報を優先
- 客観的事実に基づく内容
【形式】
1. 概要（100字）
2. 主要なトピック（3-5項目）
3. 重要な数値データ
4. 今後の展望
5. 情報源の明記
```

**実践例: AI市場調査**
```
AI市場調査の専門家として、「2024年の生成AI市場動向」について、
以下の項目で最新情報をまとめてください：

【調査項目】
- 市場規模と成長率
- 主要企業の動向（Google、OpenAI、Microsoft等）
- 新技術トレンド
- 規制動向
- 今後の予測

【出力形式】
- エグゼクティブサマリー（200字）
- 各項目の詳細（項目ごとに300字程度）
- データは可能な限り数値で示す
- 情報源を明記
```

#### 2. 要約プロンプト

**段階別要約テクニック:**

*レベル1: 基本要約*
```
以下の文章を3つのポイントに要約してください：
[文章]
```

*レベル2: 構造化要約*
```
以下の文章を読み、下記の形式で要約してください：
【主張】（1文）
【根拠】（3つのポイント）
【結論】（1文）
【重要度】（高/中/低）

[文章]
```

*レベル3: 対象者別要約*
```
以下の技術文書を、非エンジニアの経営陣向けに要約してください：
【目的】投資判断のための情報提供
【制約】専門用語を使わず、ビジネスインパクトに焦点
【文字数】300字以内

[文章]
```

#### 3. 翻訳プロンプト

**高品質翻訳のコツ:**

*基本翻訳:*
```
以下の文章を自然な日本語に翻訳してください。
専門用語は適切な日本語表記を使用し、
文脈に応じて意訳も行ってください：

[英文]
```

*文書種別対応翻訳:*
```
【翻訳タイプ】ビジネス文書
【対象読者】日本企業の役員
【注意点】
- 丁寧語を使用
- カタカナ表記は最小限に
- 日本のビジネス慣習に適応

以下のメールを翻訳してください：
[英文メール]
```

#### 4. アイデア発想プロンプト

**創造性を引き出すテクニック:**

*ブレインストーミング:*
```
【役割】クリエイティブディレクター
【課題】[具体的な課題]
【制約】予算[金額]、期間[期間]
【目標】従来にない斬新なアプローチ

上記条件で、10個のアイデアを発想してください。
各アイデアは以下の形式で：
- アイデア名
- 概要（50字）
- 実現可能性（5段階評価）
- 期待効果
```

*SCAMPER法の活用:*
```
以下の既存商品を、SCAMPER法で改良アイデアを考えてください：

商品：[商品名]
S（代替）：何を代わりに使えるか？
C（結合）：何と組み合わせられるか？
A（適応）：何を真似できるか？
M（修正）：何を変更できるか？
P（転用）：他の用途はあるか？
E（除去）：何を取り除けるか？
R（逆転）：逆にしたらどうなるか？

各項目で2-3のアイデアを提示してください。
```

#### 5. 文章校正プロンプト

**多層的校正アプローチ:**

*基本校正:*
```
以下の文章を校正してください：
【チェック項目】
- 文法・語法の誤り
- 誤字・脱字
- 表記の統一
- 読みやすさの改善

【出力形式】
1. 修正後の文章
2. 修正箇所の説明
3. 改善提案

[原文]
```

*目的別校正:*
```
【文書タイプ】プレゼンテーション資料
【対象聴衆】投資家
【目的】資金調達
【文字数制限】スライド1枚あたり50字以内

上記条件で以下の文章を最適化してください：
- 簡潔で力強い表現に
- 数値データを活用
- 説得力のある構成に

[原文]
```

## 実践課題

### 課題1: ペルソナ別自己紹介文生成

3つの異なるペルソナ（役割）になりきって、AIに自己紹介文を生成させてください。

**ペルソナ例:**
1. 営業担当者（顧客向け）
2. 技術者（同業者向け）
3. 経営者（投資家向け）

**評価基準:**
- プロンプトの指示の明確さ
- 生成物の一貫性
- ペルソナの特徴の表現

### 課題2: 最新ニュースの分析・企画立案

以下の手順で課題を実行してください：

1. **情報収集**: 特定のテーマ（例：リモートワーク動向）について最新ニュースをAIで収集・要約
2. **分析**: 収集した情報から3つのトレンドを特定
3. **企画立案**: トレンドに基づいた新サービスのアイデアを3つ創出

**使用ツール**: Perplexity AI または ChatGPT + Web検索
**成果物**: 企画書（1000字程度）

**評価基準:**
- 情報収集の網羅性
- アイデアの独創性
- 実現可能性の考慮

## 応用テクニック

### Chain of Thought（思考連鎖）

複雑な問題を段階的に解決させる手法

```
以下の問題を段階的に考えて解決してください：

【問題】売上が前年同期比20%減少した原因と対策

【思考プロセス】
1. まず、売上減少の可能性要因を列挙
2. 各要因の影響度を分析
3. 最も可能性の高い原因を特定
4. その原因に対する具体的対策を立案

各ステップで思考過程を明示してください。
```

### Few-shot Learning

例示による学習効果の活用

```
以下の例に従って、商品レビューの感情分析を行ってください：

【例1】
レビュー：「素晴らしい商品です。期待以上でした。」
感情：ポジティブ（8/10）
理由：「素晴らしい」「期待以上」等の強い肯定表現

【例2】
レビュー：「普通です。可もなく不可もなく。」
感情：ニュートラル（5/10）
理由：「普通」「可もなく不可もなく」等の中立表現

【分析対象】
「商品は良いのですが、配送が少し遅れました。」
```

## よくある問題と対処法

### 問題1: 期待した回答が得られない

**原因:**
- プロンプトが曖昧
- 文脈情報が不足
- 出力形式の指定がない

**対処法:**
- 5W1Hで詳細化
- 具体例の追加
- 段階的な質問への分割

### 問題2: 情報の正確性に不安

**原因:**
- AIのハルシネーション
- 学習データの限界
- 最新情報の不足

**対処法:**
- 一次ソースの確認指示
- 複数の回答比較
- ファクトチェックの実施

### 問題3: 創造性が不足

**原因:**
- 制約が多すぎる
- 従来の発想に縛られている
- 多様性の指示不足

**対処法:**
- 制約の緩和
- 異業種の事例参照
- ランダム要素の導入
