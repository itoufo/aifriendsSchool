## 1. 案件獲得の全体像

### 案件獲得の5ステップ

```
ステップ1: 準備（1-2週間）
    ↓
ステップ2: プラットフォーム登録・整備（3-5日）
    ↓
ステップ3: 提案活動開始（継続的）
    ↓
ステップ4: 初案件獲得・納品
    ↓
ステップ5: 実績構築・単価アップ
```

#### 各ステップの詳細

| ステップ | やること | 期間目安 |
|---------|---------|---------|
| 準備 | スキル確認、ポートフォリオ作成、サービス設計 | 1-2週間 |
| 登録 | 各プラットフォームへの登録、プロフィール作成 | 3-5日 |
| 提案 | 毎日5-10件の提案を継続 | 2-4週間 |
| 初案件 | 丁寧な対応で高評価獲得 | 1-2週間 |
| 成長 | 実績を活かして単価・案件数UP | 継続 |

## まとめ: 案件獲得成功の公式

```
案件獲得 = スキル × 行動量 × 継続

・スキル: 学習を続け、常にアップデート
・行動量: 毎日の提案を習慣化
・継続: 結果が出るまで諦めない（最低3ヶ月）
```

**最初の1件が取れれば、その後は加速度的に楽になります。**

まずは今日から、ワークシートを埋めることから始めましょう。
