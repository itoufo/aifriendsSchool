## 1. このスクールについて

### AI Schoolの目的

AI Schoolは、**「AIを使って実際に稼げる人材」** を育成することを目的としています。

単なる知識の詰め込みではありません。このスクールでは：

- 実践的なスキルを身につける
- 実際の案件で使えるノウハウを習得する
- 収益化までの具体的な道筋を示す

これらを通じて、あなたが **副業から始めて、最終的には独立も視野に入れられる** レベルまで成長できるようサポートします。

### 「稼げるAI人材」になるまでのロードマップ概要

### 学習にかかる期間と時間の目安

| 目標 | 必要期間 | 週あたりの学習時間 |
|------|----------|-------------------|
| 副業で月5万円 | 1-2ヶ月 | 5-10時間 |
| 副業で月20万円 | 3-4ヶ月 | 10-15時間 |
| 本業レベル月50万円 | 6ヶ月-1年 | 15-20時間 |
| 独立月100万円+ | 1-2年 | 20時間以上 |

**重要**: これはあくまで目安です。あなたの現在のスキルレベルや学習効率によって大きく変わります。

## 2. あなたの目標を設定しよう

### 収入目標チェックリスト

まず、あなたの目標を明確にしましょう。以下から最も近いものを選んでください：

#### 月5万円を目指す方
- [ ] 本業の収入にプラスアルファが欲しい
- [ ] まずはAIで稼ぐ感覚を掴みたい
- [ ] リスクを最小限に副業を始めたい

**おすすめ**: Level 1-2を集中的に学習 → クラウドソーシングで小規模案件から開始

#### 月20万円を目指す方
- [ ] 副業収入を本格的に育てたい
- [ ] 将来的な独立の足がかりにしたい
- [ ] 特定の分野で専門性を持ちたい

**おすすめ**: Level 1-3を体系的に学習 → ポートフォリオ構築 → 直接営業開始

#### 月50万円を目指す方
- [ ] AIを使った仕事を本業にしたい
- [ ] フリーランスとして活動したい
- [ ] チームを持って仕事を受けたい

**おすすめ**: Level 1-4を習得 → 専門分野確立 → 高単価案件・継続案件獲得

#### 月100万円以上を目指す方
- [ ] AI事業で起業したい
- [ ] 法人向けコンサルティングをしたい
- [ ] 自社プロダクトを開発したい

**おすすめ**: 全Level習得 → 事業構築 → スケール戦略実行

### 現在のスキルレベル診断

以下の質問に答えて、あなたの現在地を把握しましょう：

**基礎レベル**
- [ ] ChatGPTを使ったことがある
- [ ] AIで文章を生成したことがある
- [ ] AIで画像を生成したことがある

**初級レベル**
- [ ] プロンプトを工夫して望む結果を得られる
- [ ] 複数のAIツールを使い分けられる
- [ ] AIを業務効率化に活用している

**中級レベル**
- [ ] AIを使って収益を得た経験がある
- [ ] クライアントワークでAIを活用している
- [ ] 特定分野でAI活用の専門知識がある

**上級レベル**
- [ ] AIを活用したサービスを提供している
- [ ] AIに関する指導・コンサルができる
- [ ] AIを使った事業を運営している

**診断結果**:
- 0-3個: Level 1から着実にスタート
- 4-6個: Level 2から実践中心で学習
- 7-9個: Level 3-4で専門性を磨く
- 10個以上: Level 5で事業化を目指す

### おすすめの学習パス

**パターンA: 最速収益化コース**（月5万円を最短で達成したい方）
```
Level 1 必須モジュールのみ（1-2週間）
    ↓
Level 2 ライティング特化（2-3週間）
    ↓
クラウドソーシング登録・案件応募開始
```

**パターンB: 着実成長コース**（しっかり基礎を固めたい方）
```
Level 1 全モジュール（1ヶ月）
    ↓
Level 2 全モジュール（1-2ヶ月）
    ↓
ポートフォリオ作成 → 案件獲得
    ↓
Level 3 へ進む
```

**パターンC: 専門特化コース**（特定分野で勝負したい方）
```
Level 1-2 基礎（2ヶ月）
    ↓
Level 3-4 専門分野に集中（3-6ヶ月）
    ↓
専門家としてのブランディング → 高単価案件
```

## 3. 学習を始める前の準備

### 必要なもの

**必須**
- PC（Windows/Mac どちらでも可）
  - 推奨スペック: メモリ8GB以上、安定したインターネット接続
- インターネット環境
  - 動画視聴ができる程度の速度があれば十分

**あると便利**
- デュアルモニター（作業効率が大幅アップ）
- メモアプリ（Notion、Obsidian等）
- スマートフォン（外出先での学習用）

### AIサービスのアカウント作成

学習を始める前に、以下のサービスのアカウントを作成しておきましょう：

**必須アカウント**
1. ChatGPT（OpenAI）
2. Claude（Anthropic）
3. Google アカウント（Gemini、その他Google AI サービス用）

**推奨アカウント**
4. Midjourney または DALL-E（画像生成用）
5. Notion（学習管理用）

詳しいアカウント作成手順は **[01_ai-services-guide.md](./01_ai-services-guide.md)** をご覧ください。

### 学習時間の確保方法

「時間がない」は最大の敵です。以下の方法で学習時間を確保しましょう：

**朝型の方**
- 起床後30分-1時間を学習時間に
- 頭がクリアな状態で新しい概念を学ぶ

**夜型の方**
- 帰宅後、夕食前の30分-1時間
- SNSやYouTubeの前に学習を完了させる

**隙間時間活用**
- 通勤時間にスマホで動画学習
- 昼休みの15分で復習
- 待ち時間にモバイルでAIツール実践

**週末集中型**
- 土日のどちらかで2-3時間まとめて学習
- 平日は15分の復習のみ

## 4. 効果的な学習方法

### インプット→アウトプットの黄金比

**理想的な比率: インプット3 : アウトプット7**

多くの人が陥る罠は「インプット過多」です。

```
❌ よくある失敗パターン
動画を見る → 動画を見る → 動画を見る → 「分かった気」になる → 実践できない

✅ 成功パターン
動画を見る → 即実践 → つまずく → 調べる → 再実践 → 定着
```

**実践のコツ**
- 新しい概念を学んだら、その日のうちに必ず1回使ってみる
- 「見るだけ」で終わらせない
- 失敗しても気にしない（むしろ学びのチャンス）

### 毎日の学習ルーティン例

**平日30分コース**
```
5分: 前日の復習
15分: 新しい内容のインプット
10分: 学んだことを実践
```

**平日1時間コース**
```
10分: 前日の復習・目標確認
20分: 新しい内容のインプット
25分: 実践・演習
5分: 学んだことをメモ
```

**休日2時間コース**
```
15分: 週の振り返り・復習
45分: 新しい内容のインプット
50分: 実践プロジェクト
10分: 翌週の計画
```

### 挫折しないためのコツ

**1. 小さく始める**
- 最初から完璧を目指さない
- 「毎日5分だけ」から始めてOK
- 習慣化することが最優先

**2. 仲間を作る**
- SNSで同じ目標の人をフォロー
- 学習内容をアウトプット（Twitter/X、ブログ等）
- 質問できる環境を持つ

**3. 成果を可視化する**
- 学習記録をつける
- 小さな成功も記録する
- 定期的に振り返る

**4. 目標を常に意識する**
- なぜ学んでいるのかを明確に
- 目標達成した自分をイメージ
- 収入目標を紙に書いて貼る

## 5. 収益化までの具体的ステップ

### Level 1-2完了 → 最初の案件獲得

**ステップ1: スキルの棚卸し（Level 2完了時点）**
- 何ができるようになったかリスト化
- 得意分野・苦手分野の把握
- 提供できるサービスの明確化

**ステップ2: プラットフォーム登録**
- クラウドワークス
- ランサーズ
- ココナラ
- （英語ができれば）Upwork、Fiverr

**ステップ3: プロフィール最適化**
- 何ができるかを具体的に記載
- 実績がなくても「学習歴」をアピール
- 丁寧で信頼できる印象を与える

**ステップ4: 最初の案件応募**
- 小規模案件から始める（1,000-5,000円程度）
- 実績作りと割り切る
- 丁寧な提案文で差別化

### ポートフォリオ構築の重要性

**なぜポートフォリオが重要か**
- 「何ができるか」を証明できる
- 信頼性が大幅にアップ
- 提案時の説得力が増す

**ポートフォリオに含めるもの**
1. 自己紹介・経歴
2. スキルセット
3. 実績・成果物（架空案件でもOK）
4. お客様の声（あれば）
5. 連絡先

**ポートフォリオの形式**
- Notion ページ（無料・簡単）
- 個人サイト（本格的）
- PDF資料（営業用）

### 実績の作り方

**実績がない段階での対策**

1. **自主制作**
   - 架空の企業向けに成果物を作る
   - 「サンプル」として提示可能

2. **知人・友人の手伝い**
   - 無料または格安で引き受ける
   - 実績として掲載許可をもらう

3. **モニター募集**
   - SNSで「無料モニター」を募集
   - 感想・レビューをもらう

4. **コンペ参加**
   - 採用されなくても経験になる
   - 提出物はポートフォリオに使える

## 6. 次のステップ

### Level 1から始めよう

準備ができたら、さっそくLevel 1の学習を始めましょう。

**Level 1で学ぶこと**
- AI の基礎知識
- ChatGPT、Claude などの使いこなし方
- 効果的なプロンプトの書き方
- AI を使った業務効率化

➡️ **[Level 1: AI基礎を学ぶ](../level1/README.md)**

### 困ったときの対処法

**技術的な問題**
- まずは検索（Google、ChatGPT に質問）
- 公式ドキュメントを確認
- コミュニティで質問

**学習の悩み**
- 一度立ち止まって目標を再確認
- 小さな成功体験を積み重ねる
- 完璧を求めすぎない

**モチベーション低下**
- なぜ始めたかを思い出す
- 成功者の事例を見る
- 休息も大切（燃え尽き防止）
