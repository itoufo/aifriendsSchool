# Level 1 実践演習: AIを使いこなす第一歩

![ハンズオン演習で実践力を磨こう](/images/illustrations/hands-on-practice.jpg)

## はじめに

### この演習の目的

この実践演習では、AIツールを実際に操作しながら、以下のスキルを身につけます：

- AIとの基本的な対話方法
- 効果的なプロンプト（指示文）の書き方
- 業務効率化への活用方法
- 副業で使えるレベルのアウトプット作成

**重要**: 読むだけでなく、必ず手を動かして実践してください。AIスキルは「使う」ことでしか身につきません。

### 準備するもの

以下のいずれかのアカウントを用意してください（無料版で十分です）：

| ツール | URL | 特徴 |
|--------|-----|------|
| ChatGPT | https://chat.openai.com | 最も普及、日本語対応良好 |
| Claude | https://claude.ai | 長文処理が得意、丁寧な回答 |
| Google Gemini | https://gemini.google.com | Google連携に強い |

**推奨**: 複数のツールを使い比べることで、それぞれの特徴がわかります。

### 演習の進め方

1. 各演習は独立しています。順番通りに進めることを推奨しますが、興味のあるものから始めても構いません
2. **所要時間は目安**です。焦らず自分のペースで進めてください
3. うまくいかないときは、それ自体が学びです。何が違ったのか分析しましょう
4. 各演習の最後に「振り返りポイント」があります。必ず記録を残してください

## 演習1: 初めてのAI対話（30分）

### 目標

- AIとの基本的なやり取りを体験する
- AIの得意なこと・苦手なことを理解する
- 質問の仕方による回答の違いを実感する

### 手順

#### ステップ1: 自己紹介をしてみる（5分）

まず、AIに自己紹介してみましょう。以下のプロンプトをコピーして、ChatGPTやClaudeに入力してください。

```
こんにちは。私は今日からAIの使い方を学び始めた初心者です。
よろしくお願いします。

あなたは何ができますか？簡単に教えてください。
```

**確認ポイント**:
- [ ] AIが自分の能力を説明してくれた
- [ ] 丁寧な挨拶が返ってきた
- [ ] 分かりやすい言葉で説明されていた

#### ステップ2: 簡単な質問をする（10分）

次に、以下の3つの質問を順番に試してください。

**質問A: 事実を聞く**
```
日本で一番高い山は何ですか？
```

**質問B: 説明を求める**
```
プログラミングとは何か、小学生にもわかるように説明してください。
```

**質問C: アドバイスを求める**
```
朝起きるのが苦手です。朝型の生活に変えるコツを3つ教えてください。
```

**確認ポイント**:
- [ ] 質問Aでは正確な回答が得られた
- [ ] 質問Bでは分かりやすい説明が得られた
- [ ] 質問Cでは実践的なアドバイスが得られた

#### ステップ3: 回答に対して深掘りする（10分）

ステップ2で得た回答の中で、もっと詳しく知りたい部分について追加質問をしてみましょう。

**例: 質問Cの回答が「1. 早く寝る 2. 朝日を浴びる 3. 朝食を食べる」だった場合**

```
「朝日を浴びる」について詳しく教えてください。
何時頃、どのくらいの時間浴びると効果的ですか？
曇りの日はどうすればいいですか？
```

**確認ポイント**:
- [ ] 前の会話を覚えていることが確認できた
- [ ] より具体的な情報が得られた
- [ ] 追加の質問にも適切に回答してくれた

#### ステップ4: AIの限界を体験する（5分）

AIには苦手なこともあります。以下を試してください。

```
今日の天気を教えてください。
```

```
私の会社の売上を教えてください。
```

**気づきメモ**:
AIは「リアルタイム情報」や「あなた個人の情報」は持っていません。これを理解しておくことは重要です。

### 振り返りポイント

以下の質問に答えて、記録を残しましょう：

1. AIとの対話で驚いたことは何ですか？
2. 便利だと感じた点は何ですか？
3. 「これは苦手そうだな」と感じた点は何ですか？

## 演習2: プロンプトの基本構造を学ぶ（45分）

### 目標

- 同じ目的でも、聞き方によって回答が大きく変わることを理解する
- 効果的なプロンプトの構造を身につける
- 自分でプロンプトを改善できるようになる

### プロンプトの黄金公式

効果的なプロンプトには以下の要素が含まれています：

```
【役割】+ 【状況/背景】+ 【具体的な指示】+ 【出力形式】
```

### 手順

#### ステップ1: Before/Afterで比較する（20分）

同じ目的を持つ3パターンのプロンプトを試し、回答の違いを確認してください。

**課題: ダイエットについてのアドバイスがほしい**

**パターン1: 曖昧な質問（悪い例）**
```
ダイエットについて教えて
```

**パターン2: 少し具体的（改善例）**
```
30代女性向けの、無理なく続けられるダイエット方法を教えてください。
```

**パターン3: 構造化されたプロンプト（良い例）**
```
あなたは栄養士の資格を持つダイエットアドバイザーです。

【相談者の状況】
- 32歳女性、デスクワーク中心
- 運動習慣なし
- 3ヶ月で5kg減量したい
- 過去に食事制限で失敗経験あり

【お願いしたいこと】
上記の状況を踏まえて、無理なく続けられるダイエットプランを提案してください。

【出力形式】
- 食事面のアドバイス（3つ）
- 運動面のアドバイス（3つ）
- メンタル面のアドバイス（2つ）
- 1週間のスケジュール例

注意: 極端な方法は避け、科学的根拠のあるアドバイスをお願いします。
```

**比較記録**:

| 項目 | パターン1 | パターン2 | パターン3 |
|------|-----------|-----------|-----------|
| 具体性 |  |  |  |
| 実用性 |  |  |  |
| 自分に合った内容か |  |  |  |

#### ステップ2: 自分でプロンプトを改善する（15分）

以下の「悪いプロンプト」を、黄金公式を使って改善してください。

**悪いプロンプト**:
```
英語の勉強法を教えて
```

**改善の観点**:
- 誰が（学習者の状況）
- 何のために（目的）
- どんな条件で（制約）
- どんな形式で（出力形式）

**あなたの改善版を書いてみましょう**:
```
（ここに自分で考えた改善版を入力してテストする）
```

#### ステップ3: テンプレートを使ってみる（10分）

以下の汎用テンプレートを使って、自分の質問を作成してください。

**汎用プロンプトテンプレート**:
```
【あなたの役割】
あなたは[専門分野]の専門家です。

【私の状況】
- 現状: [今の状態]
- 目標: [達成したいこと]
- 制約: [時間/予算/条件など]

【お願いしたいこと】
[具体的に何をしてほしいか]

【希望する出力形式】
- [箇条書き/表/ステップバイステップなど]
- 文量: [○○文字程度/簡潔に/詳しくなど]

【注意点】
- [避けてほしいこと]
- [必ず含めてほしいこと]
```

**練習課題**: このテンプレートを使って「転職活動のアドバイス」を求めるプロンプトを作成し、実際にAIに入力してみてください。

### 振り返りポイント

1. プロンプトの「何が」回答の質を変えると感じましたか？
2. 今後使えそうなテクニックは何ですか？
3. まだ難しいと感じる点は何ですか？

## 演習3: 業務効率化シミュレーション（60分）

### 目標

- 実際の業務でAIを活用するイメージを掴む
- よくある業務タスクをAIで効率化する方法を学ぶ
- 時間短縮効果を実感する

### シナリオ別演習

#### シナリオA: ビジネスメール作成（15分）

**状況**: 取引先への納期遅延のお詫びメールを書く必要がある

**プロンプト例**:
```
あなたはビジネスメールの専門家です。

以下の状況に合ったお詫びメールを作成してください。

【状況】
- 宛先: 株式会社ABC 田中様（担当者）
- 内容: 発注いただいた部品Xの納品が1週間遅れる
- 理由: 製造工程でのトラブル
- 対応: 代替品の無償提供を提案したい
- 関係性: 3年来の取引先、良好な関係

【メールの要件】
- 誠意が伝わるトーン
- 簡潔だが必要な情報は網羅
- 具体的な対応策を明記
- 今後の対策にも触れる

【出力形式】
件名と本文を分けて出力してください。
```

**やってみよう**:
1. 上記プロンプトをコピーして実行
2. 生成されたメールを確認
3. 修正したい点があれば「〇〇の部分をもっと丁寧に」などと追加指示

**時間測定**:
- 自分で一から書いた場合の想定時間: ___分
- AIを使った場合の実際の時間: ___分

#### シナリオB: 議事録の要約（20分）

**状況**: 長い会議のメモから、要点をまとめた議事録を作成する

**練習用の会議メモ**:
```
【会議メモ】
日時: 2024年1月15日 14:00-16:00
参加者: 山田部長、佐藤課長、鈴木、田中、高橋

- 山田部長から第3四半期の売上報告。前年比105%で目標達成。
- 特に新製品Aが好調、想定の120%の売上。
- 一方で製品Bは苦戦、価格競争が激しい。
- 佐藤課長から来期の戦略案。製品Aへの投資強化を提案。
- 鈴木からマーケティング予算の増額要望。SNS広告を強化したい。
- 田中は人員増強を要望。現状の人数では新製品対応が困難。
- 高橋から顧客からのフィードバック報告。品質には満足だがサポート対応への不満あり。
- サポート体制の強化について議論。外注vs内製で意見分かれる。
- 結論: 来週までに各自で対策案を持ち寄る。
- 次回会議は1月22日15時から。
```

**プロンプト例**:
```
以下の会議メモを、ビジネス文書として適切な議事録に整形してください。

【整形のポイント】
1. 日時・参加者・議題を明記
2. 討議事項を整理して箇条書き
3. 決定事項とアクションアイテムを明確に
4. 次回予定を記載

【会議メモ】
（上記のメモを貼り付け）

【出力形式】
正式な議事録のフォーマットで出力してください。
```

**やってみよう**:
1. プロンプトを実行
2. 生成された議事録を確認
3. 「アクションアイテムを表形式に」など追加指示で調整

#### シナリオC: 調査レポート作成（25分）

**状況**: 上司から「在宅勤務のメリット・デメリットをまとめて」と依頼された

**プロンプト例**:
```
あなたはビジネスリサーチャーです。

「在宅勤務（リモートワーク）のメリットとデメリット」について、
上司に報告するための簡潔なレポートを作成してください。

【レポートの要件】
- 対象読者: 中小企業の経営層
- 目的: 在宅勤務制度導入の検討材料
- 視点: 企業側と従業員側の両面から

【構成】
1. 概要（100字程度）
2. メリット（企業側3つ、従業員側3つ）
3. デメリット（企業側3つ、従業員側3つ）
4. 導入時の注意点（3つ）
5. まとめ

【出力形式】
- 各項目は簡潔な箇条書き
- 専門用語は避け、分かりやすい表現で
- 全体で800字程度
```

**発展課題**: 生成されたレポートに対して、以下の追加指示を試してみましょう。

```
このレポートをPowerPointのスライド構成に変換してください。
1スライドあたり3つのポイントに絞って、プレゼン用に整理してください。
```

## 演習4: 副業を意識した実践課題（90分）

### 目標

- 実際に「納品物」として使えるクオリティのアウトプットを作成する
- クライアントワークを想定した品質チェックを行う
- AIライティングの実践スキルを身につける

### 課題A: ブログ記事作成（30分）

**シナリオ**: Webライターとして「初めての一人暮らし準備ガイド」という記事を依頼された

**プロンプト例**:
```
あなたはSEOとユーザー体験を熟知したWebライターです。

【記事の依頼内容】
- タイトル: 「初めての一人暮らし準備ガイド」
- ターゲット読者: 18-22歳の大学生・新社会人
- 記事の目的: 一人暮らしの不安を解消し、準備の全体像を把握させる
- 文字数: 2000-2500文字
- トーン: 親しみやすく、先輩が後輩に教えるような雰囲気

【構成の指定】
1. リード文（読者の悩みに共感）
2. 一人暮らしに必要な準備リスト
3. 引っ越し前にやること（時系列で）
4. 初期費用の目安と節約術
5. 先輩からのアドバイス
6. まとめ

【SEO要件】
- キーワード「一人暮らし 準備」を自然に含める
- 見出しは読者の疑問形式で（例:「何から始めればいい？」）
- 箇条書きやリストを活用して読みやすく

【注意点】
- 具体的な金額は「目安」として記載
- 地域差があることに触れる
- 読者が次のアクションを取れるように締める
```

**やってみよう**:
1. プロンプトを実行して記事を生成
2. 以下のチェックリストで品質確認
3. 必要に応じて修正指示

**品質チェックリスト**:
- [ ] ターゲット読者に合った言葉遣いになっているか
- [ ] 見出しが魅力的で内容を予測できるか
- [ ] 具体的な数字や例が含まれているか
- [ ] 読者が次に何をすべきか明確か
- [ ] 誤字脱字がないか
- [ ] 文章の流れが自然か

### 課題B: リサーチレポート作成（30分）

**プロンプト例**:

## 演習5: 自分の業務を効率化する（自由課題）

### 目標

- 学んだスキルを自分の環境に適用する
- 実際に時間短縮を実感する
- 継続的にAIを活用する習慣をつくる
